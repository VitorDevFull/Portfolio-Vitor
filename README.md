# Portfólio Pessoal - Vitor Cruz do Nascimento

Este é meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e formas de contato.

O objetivo deste projeto é reunir em uma única página meus principais projetos de estudo e prática em desenvolvimento web, utilizando HTML, CSS e JavaScript.

## 🔗 Acesse o projeto

[Ver Portfólio Online](https://vitordevfull.github.io/Portfolio-Vitor)

## 📌 Sobre o projeto

O portfólio contém seções de apresentação, sobre mim, projetos, habilidades e contato.

Na seção de projetos, é possível acessar alguns dos meus projetos desenvolvidos durante meus estudos:

- Lista de Tarefas
- Calculadora Simples
- Jogo da Velha

## 💻 Funcionalidades

- Página responsiva
- Menu de navegação por seções
- Cards de projetos clicáveis

- Formulário de contato com envio para WhatsApp
- Links para GitHub e LinkedIn

## 📚 Aprendizados

Neste projeto, pratiquei a criação de uma página web completa utilizando HTML, CSS e JavaScript.

Também pratiquei a organização de arquivos em pastas, criação de layout responsivo, uso de links externos, integração com WhatsApp e publicação de projeto no GitHub Pages.

## 📸 Projetos apresentados

- **Lista de Tarefas:** aplicação para adicionar, concluir e remover tarefas, utilizando HTML, CSS, JavaScript e localStorage.
- **Calculadora Simples:** calculadora desenvolvida com JavaScript, utilizando manipulação do DOM e lógica de operadores matemáticos.
- **Jogo da Velha:** jogo desenvolvido com HTML, CSS e JavaScript, com alternância de jogadores, verificação de vitória, empate e placar.
