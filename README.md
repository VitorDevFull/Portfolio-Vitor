# Portfólio Pessoal - Vitor Cruz do Nascimento

Este é meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e formas de contato.

O objetivo deste projeto é reunir em uma única página meus principais projetos de estudo e prática em desenvolvimento web, utilizando HTML, CSS e JavaScript.

## 🔗 Acesse o projeto

## 📌 Sobre o projeto

O portfólio contém seções de apresentação, sobre mim, projetos, habilidades e contato.

Na seção de projetos, é possível acessar alguns dos meus projetos desenvolvidos durante meus estudos, como:

- Lista de Tarefas

📚 Aprendizados

Neste projeto, pratiquei a criação de uma página web completa utilizando HTML, CSS e JavaScript.

Também pratiquei organização de arquivos em pastas, criação de layout responsivo, uso de links externos, integração com WhatsApp e publicação de projeto no GitHub Pages.
